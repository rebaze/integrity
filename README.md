# Integrity Library

- Composite hash- tree based java library (DAG)
- OSGi compatible
- Embeddable
- Low footprint 

Small Java library implementing Merkle Trees (https://en.m.wikipedia.org/wiki/Merkle_tree).

## What is this good for

- Blockchain Technologies
- Indexing
